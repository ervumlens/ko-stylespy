The Basics

Style Spy is an extension for Komodo IDE and Komodo Edit. It simplifies debugging and prototyping styles used in the Komodo editor.

Installation

The extension XPI and source are available to download on this page.

Once installed, the extension adds a new Style Spy menu under the main Tools menu.

Build

Building Style Spy requires the CoffeeScript compiler available from the environment PATH and an installation of Komodo Edit or IDE. The ko-stylespy repository includes Komodo macros that build the extension. Just clone the repo, open a new Komodo project from within the ko-stylespy directory, and run the macros from the Komodo toolbox.
